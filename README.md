# 🛒 EzzyStore – Online Grocery Store Website

EzzyStore is a fully functional, user-friendly online grocery shopping website that offers seamless shopping experiences, real-time cart and wishlist management, and secure backend functionality using PHP and MySQL.

---

## 🚀 Features

- 🔐 **User Authentication**
  - Sign up and login system with real-time input validation
  - Secure session management

- 🛍️ **Shopping Experience**
  - Browse a wide range of grocery items with price and offers
  - Add items to Cart and Wishlist dynamically
  - Quantity updates and item removal from cart

- 👤 **Profile Management**
  - View and update user details like name, email, gender, address, and phone number

- 📦 **Order System**
  - Delivery address form with data storage in MySQL
  - Realistic payment page with:
    - Card type detection (Visa/MasterCard)
    - Expiry date & CVV validation
    - Masked CVV input
    - Spinner during submission
  - OTP verification via email with resend functionality

- 📜 **Order History & Tracking**
  - View past orders and track current ones

- 💌 **Feedback System**
  - Integrated feedback form to collect user reviews
